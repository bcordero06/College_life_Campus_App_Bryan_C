# College Life Campus App
This college life campus app website was simple web application, designed to help of student needs on wanting to know more of campus events, diniing options and be able to navigate campus with ease.

# Features

### Dashboard
Displays the current weather of Chicago using open metro api weather,actions and the campus status updates.

### Quick Actions
Fats acess to Events,Dining hours,campus map and issue form page.

### Events Page
Shows upcoming Events with images and search bar, plus view details button that leads to the event website about morr information

### Dining Page
Shows diffrent dining locations around campus and hours plus a link that diercts to the menus.

### Campus Map
Dispalys the campus map on the diffrent buildings around campus and where they are and also shows a embed Google map for navigation to campus.

### Report Issue Form
Easy to fill out form for students and faculty to fill out for any issues that occur in campus

# Technolgies Used
- HTML5
- CSS3
- Bootstrap 5
- Javascript
- Open Metro Api(Weather)
